Odyssey - Automatic Light Control 

When a Stage module is connected to the Lamp Controller slot of the Odyssey module in a SySal layout, it is possible to automatically adjust the light.

The relationship between the light control signal and the light output is assumed to be monotonically increasing (low ripple). Each time the automatic light control function changes the light, it waits enough for the light output to stabilize. The length of this wait depends on how much the light control value changes; Odyssey assumes the dynamic response of the lamp is an exponential relaxation, with a time constant that can be adjusted by the user.

During light control, all the image processing functions are disabled. Neither synchronous nor asynchronous Vision Processor functions run.

Automatic light control can work using two criteria.

  • Average Gray Level Control: the light is changed so that the average gray level matches the user-defined gray level as closely as possible.
  • Saturation Control: the light is changed so that the fraction of saturated pixels matches the user-defined saturation fraction as closely as possible. PRESENTLY THIS ALGORITHM IS NOT IMPLEMENTED, though the saturation fraction is always estimated.
Saturated pixels are defined as those pixels whose gray level is greater or equal to a saturation threshold; this accounts for the fact that not necessarily all pixels for a sensor cover the whole range of their ADCs.
